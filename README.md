# Application for the exchange of secure messages

Решение тестового задания


## Technologies
* Node.js
* Express
* MongoDB
* Pug Templating
* jQuery
* Bootstrap
* Method-override
* NanoID
* Simplecrypt
* Parse-error

### Version
1.0.0

## Usage


### Installation

Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```
